==============================================================================
INTRODUCTION                                   *qf-tooltip* *qf-tooltip-intro*

Display the error message for the current line in a |popup| window (like a
tooltip).

The plugin checks if the current |quickfix| list (or |location-list|) contains
any errors for the current line and displays them in a popup window at the
current cursor position.

==============================================================================
USAGE                                                       *qf-tooltip-usage*

Mappings ~

<plug>(qf-tooltip-qflist)                          *<plug>(qf-tooltip-qflist)*
            Open a |popup| window with the error message for the current line
            at the current cursor position. The plugin checks if the current
            |quickfix| list contains any errors for the current line. If the
            line contains several errors, all will be collected and listed in
            the same popup.

            Example: >
                " mnemonic: popup quickfix error
                nmap <leader>pq <plug>(qf-tooltip-qflist)
<
<plug>(qf-tooltip-loclist)                        *<plug>(qf-tooltip-loclist)*
            Same as above but uses the current |location-list|.

            Example: >
                " mnemonic: popup location-list error
                nmap <leader>pl <plug>(qf-tooltip-loclist)

Functions ~

qf#tooltip#show({what})                                    *qf#tooltip#show()*

            The above mappings use the current quickfix or location lists. If
            you want to show errors from another list (for example another
            quickfix list in the stack), use |qf#tooltip#show()| directly.

            {what} must be a |Dictionary| containing the following keys:

            "title"         |String| specifying the title for the popup
                            window.

            "items"         |List| of dictionaries, each containing the
                            following keys:
                            bufnr (buffer number)
                            lnum  (line number)
                            col   (column number)
                            text  (error message)

                            See |getqflist()| for a more detailed explanation
                            of the items.

==============================================================================
CONFIGURATION                                              *qf-tooltip-config*

The appearance of the popup window can be configured through the following
highlight groups:

QfTooltip                                                       *hl-QfTooltip*
                Popup window background and error text. This is the equivalent
                of the |hl-Normal| highlight group in regular windows.
                Default: |hl-Pmenu|

QfTooltipTitle                                             *hl-QfTooltipTitle*
                Title (top line) of popup window.
                Default: |hl-Title|

                Note: The title of the popup window is set to the
                |quickfix-title| of the current quickfix list.

QfTooltipLineNr                                           *hl-QfTooltipLineNr*
                Line and column number.
                Default: |hl-Directory|
